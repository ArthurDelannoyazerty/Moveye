Projet Moveye

Avec la caméra de l'ordinateur on détecte où le regard se pose sur l'écran et la souris s'y déplace.
Le script final est dans : projetfinal\src\projetfinal\ProjetJavaMovEye.java
Il a été fait en Terminale pour l'option ISN du Bac.

Il fonctionne avec : jdk 1.8.0_181
		     la librairie JavaFX
		     la librairie openCV 345

Le programme est compliqué à lancer à cause de ces librairies qui doivent prendre des chemains bien précis.

Il faut aussi le lancer avec l'IDE ecplipse normalement.
